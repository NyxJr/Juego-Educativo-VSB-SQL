# Juego-Educativo-VSB-SQL
Juego de adivinanza  basado en VSB  + SQL   Que corra en Microsoft Basic 

"CiberAdivinanza"

Es un juego educativo  diseñado como herramienta de refuerzo para la Alfabetización Digital en alumnos de 1er año de Secundaria. Adopta el formato de adivinanza/ahorcado, donde los jugadores deben descubrir términos clave de informática (como Hardware, Sistema Operativo, Bit) basándose en una pista.

El objetivo es refrescar y consolidar conceptos básicos de informática de una manera dinámica y atractiva.

Se Busca incentivar el estudio del vocabulario técnico esencial para el inicio de la formación digital.

Estructura General:

Inicio: Un formulario de presentación capta el nombre del jugador y lo registra en la base de datos (BD). y se explica las reglas.


Juego: En el formulario principal, se cargan conceptos aleatorios. El jugador tiene hasta 6 intentos Para todo el juego.

Cada palabra incorrecta descuenta 1 intento. 
 El juego tiene la opcion para omitir la  peregunta pero incorpora un sistema de penalización -30 puntos.
Si acierta se suman 10 puntos.

Final: Al terminar,se guarda el puntaje final.
 y se muestra Tablero de Líderes: con los puntajes historicos (5). y un gracias por jugar . 

Si se le termina los intentos a mitad del juego se finaliza  y sale un cartel que lo envia a jugar nuevamente.
